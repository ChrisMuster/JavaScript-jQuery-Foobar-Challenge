The Foobar Task

The task: Enter in any number to count up to (although for demo purposes, to show it is working fully enter 15 or higher).
A list of numbers up to your choice will appear.
If a number is divisible by 3, the word "Foo" will appear instead of the number.
If a number is divisible by 5, the word "Bar" will appear instead of the number.
If the number is divisible by both 3 and 5, "FooBar" will appear instead of the number.
All other numbers will appear as normal.


Reminder

Your input must be a number greater than zero.
